# 🧠 Shadow Vote Arena

**Let the Internet decide your fate.**

Anonymous War Room is a chaotic, real-time platform where users drop wild personal dilemmas like:

- "Should I quit my job?"
- "Should I text my ex?"
- "Should I spend all my savings on a one-way ticket?"

...and the internet votes.  
**Majority wins. No takebacks.**

## 💡 Idea

I was bored. I'm a coder. I wanted drama.

So I built a platform where strangers anonymously vote on your life decisions in real-time — with live chat, community chaos, and no accountability.

## 🚀 Features

- ✅ Submit anonymous dilemmas
- 🗳️ Real-time voting (Majority rules!)
- 💬 Live chat room per vote — witness the drama
- 🔥 Trending questions board
- ⌛ Timer-based decision locks
- 🧨 Community-powered chaos

## 🛠️ Tech Stack

- **Frontend**: React + TailwindCSS
- **Backend**: Node.js / Express
- **Real-Time**: Socket.IO / Firebase / Supabase (TBD)
- **Database**: MongoDB / PostgreSQL
- **Auth**: Anonymous (optionally wallet-based via Web3)
- **Deployment**: Vercel / Render / Railway
